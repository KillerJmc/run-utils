# run-utils



## 描述

此工具可让您通过双击运行特定编程语言的文件。



## 用法

1. 将此仓库克隆到磁盘上的任何位置。
2. 双击`Install.cmd`来安装此软件。
3. 现在你就可以双击运行.c, .cpp, .java, .groovy, .lua, .py文件了！



## 注意

1.您需要将以下项目定义为系统环境变量：

+ **MINGW_HOME**：C，Cpp

+ **JAVA_HOME**：Java、Groovy

+ **PYTHON_HOME**：Python
