# run-utils cpp

## 描述

此工具可让您通过双击运行单个C++语言脚本！

## 用法

1. 将 **MINGW_HOME** 定义为系统环境变量。
2. 将此仓库克隆到磁盘上的任何位置。
3. 双击.cpp，指定 `bin/RunCpp.exe` 作为默认打开程序。
4. 尽情享受吧！
