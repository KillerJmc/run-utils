# run-utils lua

## Description

This tool lets you run a single lua script by double-click!

## Usage

1. Clone this repository to anywhere on the disk you like.
2. Double-click a .lua file, specify `bin/RunLua.exe` as the default program to open.
3. Enjoy it!
